# Wind 功能清单

## 已完成功能

### 1. AI 智能排版 ✅

**功能描述**：使用 AI 模型对文章进行智能排版，优化格式和结构。

**实现方式**：

- 调用 GLM-4.7 Flash API
- 使用优化的系统提示词（system_prompt.md）
- 支持自定义提示词输入

**核心特性**：

- 自动添加标题序号
- 优化段落结构
- 强调重点内容
- 生成列表和表格
- 添加风险提示、作者信息等标准模块

**配置方式**：

```env
AI_API_URL=https://open.bigmodel.cn/api/paas/v4/chat/completions
AI_API_KEY=your_api_key
AI_MODEL=glm-4.7-flash
```

**使用场景**：

- 公众号文章排版
- 技术文档优化
- 博客文章格式化

---

### 2. 图床上传功能 ✅

**功能描述**：将图片上传到 GitHub 仓库，并生成 CDN 加速链接。

**支持平台**：

- GitHub（已验证）

**核心特性**：

- 自动上传图片
- 生成 CDN 加速链接（jsDelivr）
- 按日期组织文件结构
- 支持自定义存储路径

**配置方式**：

```env
GITHUB_TOKEN=your_github_token
GITHUB_OWNER=kgipsonmymail
GITHUB_REPO=image-home
GITHUB_BRANCH=main
GITHUB_PATH=wind-assets
GITHUB_CDN=true
```

**文件结构**：

```
wind-assets/
├── 2026/
│   └── 03/
│       ├── timestamp1_image1.png
│       └── timestamp2_image2.png
```

**CDN 链接格式**：

```
https://cdn.jsdelivr.net/gh/{owner}/{repo}@{branch}/{path}/{file}
```

---

### 3. Markdown 渲染 ✅

**功能描述**：基于 doocs/md 渲染引擎，实现 Markdown 到 HTML 的转换。

**核心特性**：

- 支持标准 Markdown 语法
- 支持扩展语法（Mermaid、PlantUML、LaTeX 等）
- 渲染即所得
- 无损复制到公众号

**支持的扩展语法**：

- Mermaid 流程图
- PlantUML 图表
- LaTeX 数学公式
- Infographic 信息图
- Ruby 注音
- 自定义样式

**使用方式**：

1. 在编辑器中输入 Markdown 文本
2. 实时预览渲染结果
3. 一键复制到公众号

---

### 4. 排版结果预览 ✅

**功能描述**：实时预览 AI 排版后的文章效果。

**核心特性**：

- 实时渲染预览
- 支持滚动同步
- 显示渲染后的 HTML
- 支持复制 Markdown 和 HTML

**使用流程**：

1. 输入原始文章
2. 调用 AI 排版
3. 查看预览结果
4. 复制到公众号

---

### 5. 一键复制到公众号 ✅

**功能描述**：将排版后的内容一键复制到公众号编辑器。

**核心特性**：

- 复制 Markdown 源码
- 复制渲染后的 HTML
- 保持格式完整
- 支持图片链接

**使用方式**：

- 点击"复制 Markdown"按钮
- 粘贴到公众号编辑器
- 格式自动保留

### 6. 网页导入解析增强 ✅

**功能描述**：从网页（如微信公众号）导入内容时，更准确地还原样式。

**核心特性**：

- **CSS 样式鲁棒提取**：支持提取 `color`、`font-size`、`background` 等内联样式。
- **级联优先级处理**：正确识别多个同名属性，并尊重 `!important` 标志。
- **自动清理**：在转换过程中自动清除不必要的 CSS 标记，保证 Markdown 兼容性。

**相关文件**：

- `apps/web/src/wind/core/html-parser.ts`

4. **浏览器兼容性**：部分高级特性可能不支持旧浏览器
