# Wind 项目目录结构

## 项目根目录

```
md/
├── apps/                    # 应用程序目录
│   ├── web/                # Web 应用
│   │   ├── src/            # 源代码
│   │   ├── public/         # 静态资源
│   │   ├── package.json    # 依赖配置
│   │   └── vite.config.ts  # Vite 配置
│   ├── vscode/             # VS Code 扩展
│   └── utools/             # uTools 插件
├── packages/               # 共享包
│   ├── core/              # 核心渲染引擎
│   ├── shared/            # 共享工具和类型
│   └── md-cli/            # 命令行工具
├── wind/                   # Wind 中间件（我们的开发重点）
│   ├── docs/              # 文档目录
│   ├── system_prompt.md   # AI 排版系统提示词
│   ├── 初始语法.md        # Markdown 语法参考
│   └── prompt.txt         # 开发记录
├── docs/                   # 项目文档
├── scripts/                # 构建脚本
├── public/                 # 公共资源
├── docker/                 # Docker 配置
├── package.json           # 根项目配置
├── pnpm-workspace.yaml    # PNPM 工作区配置
└── tsconfig.json          # TypeScript 配置
```

## Wind 中间件详细结构

```
wind/
├── docs/                   # 文档目录
│   ├── 项目结构.md        # 本文件
│   ├── 功能清单.md        # 已完成功能清单
│   ├── 错题本.md          # Bug 记录和解决方案
│   ├── 提示词记录.md      # Prompt 提示词记录
│   ├── 开发进展.md        # 开发进展文档
│   ├── README.md          # 项目简介
│   └── 代码规范.md        # 代码规范说明
├── system_prompt.md        # AI 排版系统提示词（核心）
├── 初始语法.md             # Markdown 公众号渲染语法参考
└── prompt.txt              # 开发记录和待整理内容
```

## 核心文件说明

### AI 排版相关

- **system_prompt.md**：AI 排版的系统提示词，包含完整的排版规则和示例
- **初始语法.md**：Markdown 公众号渲染语法参考文档

### 文档相关

- **开发进展.md**：记录项目开发历程和未来计划
- **功能清单.md**：已完成功能的详细说明
- **错题本.md**：开发过程中遇到的 Bug 和解决方案
- **提示词记录.md**：Prompt 提示词的迭代历史
- **README.md**：项目简介和快速开始指南
- **代码规范.md**：开发过程中的代码规范要求

### 配置文件

- **.env**：环境变量配置（不提交到 Git）
  - AI 服务配置
  - 图床配置
  - 其他敏感信息

## Web 应用结构

```
apps/web/
├── src/                    # 源代码
│   ├── components/        # 组件
│   ├── views/            # 页面视图
│   ├── utils/            # 工具函数
│   ├── stores/           # 状态管理
│   └── main.ts           # 入口文件
├── public/                 # 静态资源
│   ├── wind/             # Wind 相关资源
│   │   └── system_prompt.md  # 系统提示词副本
│   └── upload/           # 上传文件目录
├── package.json           # 依赖配置
├── vite.config.ts         # Vite 配置
└── .env                   # 环境变量
```

## 核心包结构

### packages/core/

核心渲染引擎，负责 Markdown 到 HTML 的转换。

```
packages/core/
├── src/
│   ├── renderer/         # 渲染器
│   ├── theme/           # 主题系统
│   ├── extensions/      # 扩展功能
│   │   ├── mermaid.ts   # Mermaid 图表
│   │   ├── plantuml.ts  # PlantUML 图表
│   │   ├── katex.ts     # LaTeX 公式
│   │   └── infographic.ts # 信息图
│   └── utils/           # 工具函数
└── package.json
```

### packages/shared/

共享工具和类型定义。

```
packages/shared/
├── src/
│   ├── types/           # 类型定义
│   ├── utils/           # 工具函数
│   ├── configs/         # 配置文件
│   └── editor/          # 编辑器相关
└── package.json
```

## 文档引用关系

```
README.md (项目简介)
    ├── 引用 → 项目结构.md
    ├── 引用 → 功能清单.md
    ├── 引用 → 开发进展.md
    └── 引用 → 代码规范.md

开发进展.md
    ├── 引用 → 功能清单.md
    ├── 引用 → 错题本.md
    └── 引用 → 提示词记录.md
```

## 注意事项

1. **Wind 目录**：我们的开发重点，所有中间件相关代码和文档都放在这里
2. **文档集中管理**：所有文档存放在 wind/docs 目录，避免重复
3. **环境变量**：敏感配置信息存放在 .env 文件，不提交到 Git
4. **引用关系**：文档之间通过引用关系组织，避免内容重复
