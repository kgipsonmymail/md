# Wind 项目简介

## 项目概述

Wind 是基于 doocs/md 项目开发的中间件，专注于公众号文章的 AI 智能排版。项目目标是提供一套完整的文章排版解决方案，包括 Markdown 语法转译、渲染预览、公众号兼容导出和自动上传等功能。

## 核心功能

### 1. AI 智能排版 ✅

使用 AI 模型对文章进行智能排版，优化格式和结构。

- 自动添加标题序号
- 优化段落结构
- 强调重点内容
- 生成列表和表格
- 添加标准模块（风险提示、作者信息等）

### 2. 图床上传功能 ✅

将图片上传到 GitHub 仓库，并生成 CDN 加速链接。

- 自动上传图片
- 生成 CDN 加速链接
- 按日期组织文件结构
- 支持自定义存储路径

### 3. Markdown 渲染 ✅

基于 doocs/md 渲染引擎，实现 Markdown 到 HTML 的转换。

- 支持标准 Markdown 语法
- 支持扩展语法（Mermaid、PlantUML、LaTeX 等）
- 渲染即所得
- 无损复制到公众号

### 4. 排版结果预览 ✅

实时预览 AI 排版后的文章效果。

- 实时渲染预览
- 支持滚动同步
- 显示渲染后的 HTML

### 5. 一键复制到公众号 ✅

将排版后的内容一键复制到公众号编辑器。

- 复制 Markdown 源码
- 复制渲染后的 HTML
- 保持格式完整

## 快速开始

### 环境要求

- Node.js >= 22.16.0
- pnpm >= 8.0.0

### 安装依赖

```bash
pnpm install
```

### 配置环境变量

创建 `.env` 文件：

```env
# AI 服务配置
AI_API_URL=https://open.bigmodel.cn/api/paas/v4/chat/completions
AI_API_KEY=your_api_key
AI_MODEL=glm-4.7-flash

# 图床配置（GitHub）
GITHUB_TOKEN=your_github_token
GITHUB_OWNER=kgipsonmymail
GITHUB_REPO=image-home
GITHUB_BRANCH=main
GITHUB_PATH=wind-assets
GITHUB_CDN=true
```

### 启动开发服务器

```bash
pnpm start
```

访问 http://localhost:5173

## 使用指南

### 1. AI 排版

1. 在编辑器中输入原始文章
2. 点击"AI 排版"按钮
3. 等待 AI 处理完成
4. 查看排版结果

### 2. 图片上传

1. 点击"上传图片"按钮
2. 选择图片文件
3. 等待上传完成
4. 复制图片链接

### 3. 复制到公众号

1. 完成排版后
2. 点击"复制 Markdown"按钮
3. 粘贴到公众号编辑器
4. 格式自动保留

## 核心特性

### 统一的排版风格

基于优化的系统提示词，确保所有文章采用统一的排版风格，适合公众号系列文章。

### 智能内容理解

AI 理解文章内容，自动添加标题序号、子标题、列表等，提升文章可读性。

### 渲染即所得

基于 doocs/md 渲染引擎，确保预览效果与公众号实际效果一致。

### 无损复制到公众号

支持一键复制到公众号编辑器，格式完整保留，无需手动调整。

## 性能指标

- AI 排版响应时间：~5-10 秒
- 图片上传速度：~2-5 秒/张
- 渲染预览速度：实时

## 已知限制

1. **AI 排版质量**：依赖于 AI 模型，可能存在不稳定
2. **图片大小限制**：GitHub 单文件限制 100MB
3. **API 调用限制**：GLM-4 API 有调用频率限制
4. **浏览器兼容性**：部分高级特性可能不支持旧浏览器
