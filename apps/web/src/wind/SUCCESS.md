## 🚀 立即使用

### 1. 访问应用

打开浏览器访问：

```
http://localhost:5173/md/
```

### 2. 找到 Wind 按钮

在页面右下角，你会看到一个**绿色的圆形按钮**（🎨 图标）

### 3. 打开 Wind 处理器

点击绿色按钮，Wind 处理器会从右侧滑入

### 4. 配置 Wind

首次使用需要配置：

1. 点击"显示配置"按钮
2. 填入配置信息：

**GitHub 配置**（用于图床）

- Token: 访问 https://github.com/settings/tokens 获取
  - 点击 "Generate new token (classic)"
  - 勾选 `repo` 权限
  - 生成并复制（格式：`ghp_xxxxxxxxxxxx`）
- 用户名: 你的 GitHub 用户名
- 仓库名: 创建一个仓库用于存储图片（如 `image-hosting`）
- 分支名: `main`
- 存储路径: `wind-assets`
- ✓ 使用 jsDelivr CDN 加速

**AI 配置**（用于智能排版）

- API 地址: `https://api.openai.com/v1/chat/completions`
- API 密钥: 访问 https://platform.openai.com/api-keys 获取
  - 创建 API Key（格式：`sk-xxxxxxxxxxxx`）
- 模型: `gpt-4` 或 `gpt-3.5-turbo`
- Temperature: `0.7`

3. 点击保存（配置会自动保存到 localStorage）

### 5. 开始使用

Wind 支持三种模式：

#### 模式 1：上传 .docx 文件

1. 点击"上传 .docx"标签
2. 拖拽或选择 Word 文档
3. 勾选"上传图片到图床"和"AI 智能排版"
4. 点击"开始处理"
5. 等待处理完成
6. 点击"复制 Markdown"

#### 模式 2：文本 + 图片

1. 点击"文本 + 图片"标签
2. 粘贴或输入文章内容
3. 点击"选择图片"上传图片
4. 点击"开始处理"
5. 复制结果

#### 模式 3：仅 AI 排版

1. 点击"仅 AI 排版"标签
2. 粘贴 Markdown 内容
3. 点击"开始处理"
4. 复制优化后的内容

## 🐛 故障排除

### 问题 1：看不到绿色按钮

**解决方案**：

- 刷新页面（Ctrl + F5）
- 检查浏览器控制台是否有错误

### 问题 2：点击按钮没反应

**解决方案**：

- 打开浏览器控制台（F12）查看错误
- 确认 Wind 组件路径正确

### 问题 3：配置保存失败

**解决方案**：

- 检查浏览器是否允许 localStorage
- 清除浏览器缓存后重试

### 问题 4：图片上传失败

**解决方案**：

- 检查 GitHub Token 是否有 `repo` 权限
- 检查仓库名和用户名是否正确
- 检查网络连接

### 问题 5：AI 排版失败

**解决方案**：

- 检查 API Key 是否正确
- 检查 API 余额是否充足
- 检查网络连接
