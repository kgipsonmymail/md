# Wind 功能模块清单

## 概述

Wind 是一个智能排版中间件，提供 AI 排版、图床集成、文档处理等功能。

## 已完成功能

### 1. GitHub 图床集成 ✅

**功能描述**：集成 GitHub 作为图床，支持图片上传和管理。

**核心特性**：

- ✅ 图片上传到 GitHub 仓库
- ✅ 自动生成图片 URL（原始 URL 和 CDN URL）
- ✅ 基于文件哈希的本地缓存
- ✅ 云端判重：检查 GitHub 仓库中是否已存在相同文件
- ✅ CDN 加速：支持 jsDelivr CDN
- ✅ 日期路径：按年/月组织图片存储
- ✅ 配置验证：测试 GitHub 连接和权限
- ✅ 空仓库初始化：自动创建 README.md 初始化仓库
- ✅ 错误处理：详细的错误提示和诊断信息

**实现文件**：

- `services/github-image-host.ts`：核心图床服务
- `components/WindConfig.vue`：配置界面

**使用场景**：

- 上传文档中的图片
- 批量图片上传
- 图片链接生成

- ✅ 可配置参数：模型、温度、最大 Token 数
- ✅ 错误处理：完善的错误捕获和提示

**实现文件**：

- `core/ai-formatter.ts`：AI 排版引擎

**使用场景**：

- 优化文章排版
- 提高可读性
- 统一排版风格

## 功能使用流程

### 完整流程

1. **配置阶段**：配置 AI API 和 GitHub 图床
2. **输入阶段**：选择输入模式（.docx / 文本+图片 / 仅排版）
3. **处理阶段**：自动执行解析、上传、排版等步骤
4. **输出阶段**：查看 Markdown 预览和图片列表
5. **应用阶段**：复制 Markdown 或一键替换到编辑器

### 快速流程

1. **AI 工具箱**：选中文本 → 选择操作 → 一键替换
2. **AI 对话**：输入问题 → 获取回复 → 一键替换
3. **AI 文生图**：输入描述 → 生成图片 → 一键插入

## 技术栈

- **前端框架**：Vue 3 + TypeScript
- **状态管理**：Pinia
- **UI 组件**：自定义组件 + shadcn/ui
- **编辑器**：CodeMirror 6
- **HTTP 请求**：Fetch API
- **文件处理**：File API
- **本地存储**：localStorage
