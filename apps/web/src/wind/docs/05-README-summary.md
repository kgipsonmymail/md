# Wind 项目精要说明

## 概述

Wind 是一个智能排版中间件，提供 AI 排版、图床集成、文档处理等功能，旨在帮助用户快速生成高质量的公众号内容。

---

## 核心功能

### 1. AI 智能排版

使用 AI 对 Markdown 内容进行智能排版，提高文章的可读性和美观度。

**特点**：

- 自动优化标题层级
- 合理使用格式（加粗、斜体、引用等）
- 优化段落结构
- 支持自定义 Prompt

**使用场景**：

- 快速优化文章排版
- 统一排版风格
- 提高阅读体验

---

### 2. GitHub 图床集成

集成 GitHub 作为图床，支持图片上传和管理。

**特点**：

- 自动上传图片到 GitHub
- 支持 CDN 加速（jsDelivr）
- 云端判重，避免重复上传
- 按日期组织图片存储

**使用场景**：

- 上传文档中的图片
- 批量图片上传
- 图片链接生成

---

### 3. 文档解析

支持 .docx 文件解析，自动提取文本和图片。

**特点**：

- 支持 .docx 文件格式
- 提取文档文本内容
- 提取文档中的图片
- 保持文本和图片的顺序

**使用场景**：

- 从 Word 文档导入文章
- 批量处理文档

---

### 4. AI 工具箱

提供多种 AI 文本处理工具，快速优化文本。

**特点**：

- 优化文本：改善文本通顺度
- 文章总结：提取主要观点
- 错别字纠正：纠正错别字和语法错误
- 翻译功能：中英文互译
- 自定义 Prompt：支持自定义需求

**使用场景**：

- 快速优化文本
- 翻译内容
- 纠正错误

---

### 5. AI 对话

提供 AI 对话助手，与 AI 交互获取写作建议。

**特点**：

- 对话历史：保存对话记录
- 多会话管理：支持创建和切换会话
- 引用全文：可以引用编辑器全部内容
- 快捷指令：预设常用指令

**使用场景**：

- 与 AI 交互
- 获取写作建议
- 内容优化

---

### 6. AI 文生图

使用 AI 生成图片，丰富文章内容。

**特点**：

- 文生图：根据文字描述生成图像
- 图片导航：支持多图片切换
- 图片操作：插入、下载、复制链接
- 有效期管理：图片有效期提醒

**使用场景**：

- 生成配图
- 创建插图
- 丰富文章内容

---

## 技术架构

### 核心模块

1. **AI 排版引擎**（`core/ai-formatter.ts`）

   - 调用 AI API 进行智能排版
   - 支持流式响应
   - 可配置参数

2. **文档解析器**（`core/document-parser.ts`）

   - 解析 .docx 文件
   - 提取文本和图片
   - 保持顺序

3. **图片管理器**（`core/image-manager.ts`）

   - 管理图片上传
   - 图片压缩
   - 批量处理

4. **主处理器**（`core/wind-processor.ts`）

   - 协调各个模块
   - 流程编排
   - 错误处理

5. **图床服务**（`services/github-image-host.ts`）
   - GitHub 图床集成
   - 云端判重
   - CDN 加速

### 技术栈

- **前端框架**：Vue 3 + TypeScript
- **状态管理**：Pinia
- **UI 组件**：自定义组件 + shadcn/ui
- **编辑器**：CodeMirror 6
- **HTTP 请求**：Fetch API
- **文件处理**：File API
- **本地存储**：localStorage

## 快速开始

### 1. 安装

```bash
npm install
```

### 2. 配置

在 Wind 配置界面中配置：

- AI API：端点、密钥、模型等
- GitHub 图床：用户名、仓库名、分支、路径等

### 3. 使用

选择处理模式：

- **上传 .docx**：上传 Word 文档，自动解析和排版
- **文本 + 图片**：粘贴文本，上传图片，自动排版
- **仅 AI 排版**：粘贴 Markdown 内容，仅进行排版

点击"开始处理"，等待处理完成，查看结果。

### 4. 应用

- **复制 Markdown**：复制排版后的 Markdown 内容
- **替换到编辑器**：一键替换到编辑器（需要确认）

## 总结

Wind 是一个功能完善的智能排版中间件，提供了 AI 排版、图床集成、文档处理等核心功能，帮助用户快速生成高质量的公众号内容。

**核心优势**：

1. 智能排版：使用 AI 自动优化排版
2. 图床集成：支持 GitHub 图床和 CDN 加速
3. 文档解析：支持 .docx 文件解析
4. 一键操作：支持一键替换到编辑器
5. 用户友好：完善的错误提示和操作反馈

**适用场景**：

- 公众号内容创作
- 文章排版优化
- 批量文档处理
- AI 辅助写作

**技术特点**：

1. 模块化设计：核心功能独立，易于扩展
2. TypeScript 支持：类型安全，代码质量高
3. 响应式设计：支持移动端和桌面端
4. 错误处理：完善的错误捕获和提示
5. 用户体验：友好的界面和操作流程
