## AI 排版 Prompt

### 当前使用的 Prompt

```markdown
你是一名专业的 Markdown 编辑排版助手，请对以下文章进行排版优化。

要求：

1. 使用合适的标题层级（#、##、###）
2. 合理使用加粗、斜体、引用等格式
3. 优化段落结构，提高可读性
4. 保持原文的核心内容和意思不变
5. 适当使用列表、表格等格式
6. 确保排版符合公众号的显示效果

请只输出排版后的 Markdown 内容，不要输出任何额外的解释或说明。

```

### 优化方向

1. **标题层级**：根据内容重要性使用合适的标题
2. **段落结构**：优化段落长度和间距
3. **格式使用**：合理使用加粗、斜体、引用等
4. **列表使用**：将要点转换为列表
5. **表格使用**：将结构化数据转换为表格
6. **代码块**：突出显示代码和命令
7. **引用使用**：突出重要内容

## AI 文生图 Prompt

### 默认 Prompt

用户输入的文字描述直接作为 Prompt 传递给 AI 文生图 API。

### Prompt 优化建议

1. **描述清晰**：使用具体、清晰的描述
2. **风格指定**：指定图片风格（写实、卡通、插画等）
3. **颜色指定**：指定主色调或配色方案
4. **细节描述**：添加细节描述，提高图片质量
5. **尺寸指定**：指定图片尺寸（可选）

### 示例 Prompt

```
一只可爱的橘猫坐在窗台上，阳光透过窗户照在它身上，温暖而宁静，写实风格，柔和的色调
```

## 最佳实践

### 1. Prompt 设计原则

- **角色设定**：明确 AI 的角色和职责
- **任务明确**：清晰描述需要完成的任务
- **要求具体**：列出具体的要求和限制
- **示例引导**：提供示例引导 AI 输出
- **输出限制**：限制输出格式和内容

### 2. Prompt 优化方法

1. **A/B 测试**：对比不同 Prompt 的效果
2. **用户反馈**：收集用户反馈，持续优化
3. **数据分析**：分析输出结果，找出问题
4. **迭代改进**：持续迭代优化 Prompt

### 3. Prompt 管理建议

- **版本控制**：使用版本号管理 Prompt
- **文档记录**：记录每次修改的原因
- **效果评估**：评估每次修改的效果
- **备份保存**：保存历史版本，方便回退
