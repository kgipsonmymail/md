🎉 Wind 智能排版中间件

AI 驱动的 Markdown 排版工具 + GitHub 图床管理

⚠️ 重要提示

这是一个独立的测试页面，用于快速了解 Wind 中间件的功能。

要在你的项目中使用，请按照下面的步骤操作。

🚀 快速启动指南

第 1 步：启动开发服务器

在项目根目录运行：

pnpm start

或者：

pnpm web dev

第 2 步：访问测试页面

我已经为你创建了一个测试页面：

apps/web/src/views/WindTest.vue

你需要在路由中添加这个页面，或者直接在现有页面中导入 Wind 组件。

第 3 步：配置 Wind

首次使用需要配置：

  • GitHub Token（用于图床）
  • AI API Key（用于智能排版）

点击界面上的"显示配置"按钮即可配置。

第 4 步：开始使用

Wind 支持三种模式：

  • 上传 .docx：自动解析 Word 文档
  • 文本 + 图片：分别上传并自动组合
  • 仅 AI 排版：优化现有 Markdown
